### 解题思路

- 遍历数组依次加入有序集合，然后查找严格大于和小于当前值的数量，取较小值作为价值

### 性能优化

- 无

### 复杂度分析

- 设数组长度为$n$，则有
  - 时间复杂度$O(nlogn)$
  - 空间复杂度$O(n)$
    
### 代码
```#python3
class Solution:
    def checkArithmeticSubarrays(self, nums: List[int], l: List[int], r: List[int]) -> List[bool]:

        def check():
            n = len(lst)
            diff = lst[1] - lst[0]
            for j in range(2, n):
                if lst[j] - lst[j - 1] != diff:
                    return False
            return True

        ans = []
        for i, left in enumerate(l):
            lst = nums[left: r[i] + 1]
            lst.sort()
            ans.append(check())
        return ans
```
